# Hospital Emergency Department Performance Dashboard

## Project Overview

This project analyses fictional emergency department attendance, patient-flow and staffing data to identify operational bottlenecks, periods of high demand and opportunities to improve service performance.

## Business Problem

Hospital management wants to understand emergency department demand, waiting times, triage performance, patient flow and staffing pressure.

## Tools

- MySQL
- SQL
- Power BI
- Python
- Excel
- GitHub

## Data Privacy

All data used in this project is fictional and created solely for educational and portfolio purposes.
